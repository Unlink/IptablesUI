JSON Format Help

Rules should be formatted as a JSON array with the following structure:

[
  {
    "chain": "INPUT",
    "protocol": "tcp",
    "source_ip": "192.168.1.0/24",
    "dest_ip": "",
    "port": "22",
    "action": "ACCEPT"
  },
  {
    "chain": "INPUT",
    "protocol": "tcp",
    "source_ip": "",
    "dest_ip": "",
    "port": "80",
    "action": "ACCEPT"
  }
]

Fields:

  • chain: INPUT, FORWARD, or OUTPUT
  • protocol: tcp, udp, icmp, or empty for any
  • source_ip: Source IP/network or empty
  • dest_ip: Destination IP/network or empty
  • port: Port number/range or empty
  • action: ACCEPT, DROP, or REJECT
